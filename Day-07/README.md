# Day 7 - ESP32 Programming and Relay Control

## Date

24 June 2026

## Objective

Explore advanced ESP32 programming concepts and control external devices using a relay module.

## Topics Covered

* Direct GPIO Register Access
* FreeRTOS Tasks
* Dual-Core Processing
* Hardware Timers and Interrupts
* Volatile Variables
* Digital Inputs and Outputs
* Relay Control using ESP32

## Activities Performed

1. Implemented LED blinking using direct register manipulation.
2. Created tasks pinned to Core 0 and Core 1 using FreeRTOS.
3. Worked with ESP32 hardware timers and interrupts.
4. Learned the purpose of volatile variables in concurrent programming.
5. Read digital inputs and controlled outputs.
6. Controlled a relay module using ESP32 GPIO pins.
7. Observed how ESP32 handles multitasking across both CPU cores.

## Learning Outcomes

* Understood low-level GPIO control using ESP32 registers.
* Learned how FreeRTOS manages multiple tasks.
* Explored interrupt-driven programming.
* Understood the importance of volatile variables in shared data.
* Gained practical experience controlling external hardware through a relay.

## Summary

Today's session focused on deeper ESP32 programming concepts. Along with learning multitasking and interrupt handling, I controlled a relay module using ESP32, which provided a better understanding of how microcontrollers interact with external devices.
