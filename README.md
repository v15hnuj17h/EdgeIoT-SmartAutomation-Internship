# EdgeIoT & Smart Automation Internship

This repository contains my work, notes, code, and documentation from the **10-day EdgeIoT & Smart Automation Internship** conducted by **USIZO Academy**.

Throughout the internship, I explored topics ranging from Git and web deployment to Machine Learning, ESP32 programming, IoT, PCB design, Firebase, and TinyML using Edge Impulse.

---

## Internship Timeline

| Day | Topic | Highlights |
|-----|------|------------|
| **Day 1** | Git & GitHub | Repository creation, version control, documentation using Markdown |
| **Day 2** | Portfolio & Firebase | Built a portfolio website and deployed it using Firebase Hosting |
| **Day 3** | Machine Learning Basics | Introduction to AI, ML pipeline, supervised, unsupervised and reinforcement learning |
| **Day 4** | Development Environment | Installed PlatformIO, ESP32 drivers, Java JDK and required runtimes |
| **Day 5** | Applied Machine Learning | Built a Decision Tree classifier for loan approval prediction using Google Colab |
| **Day 6** | ESP32 Fundamentals | ESP32 architecture, PlatformIO projects, GPIO, Serial communication and PWM |
| **Day 7** | Advanced ESP32 Programming | FreeRTOS, multitasking, timers, GPIO registers, relay control and concurrency |
| **Day 8** | Firebase IoT Dashboard | Controlled an ESP32 remotely using Firebase Realtime Database and a hosted web dashboard |
| **Day 9** | KiCad PCB Design | Designed schematics, PCB layouts, footprints and generated Gerber files |
| **Day 10** | Edge AI & TinyML | Collected MPU6050 motion data, trained an Edge Impulse model and deployed real-time inference on ESP32 |
