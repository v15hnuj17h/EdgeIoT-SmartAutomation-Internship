# Day 6 - ESP32 Introduction and First Practical Session

## Date

23 June 2026

## Objective

Learn the fundamentals of the ESP32 microcontroller and gain hands-on experience programming it using PlatformIO.

## Topics Covered

### ESP32 Fundamentals

* Introduction to ESP32
* ESP32 Architecture
* Features and Specifications
* GPIO Pins
* Wi-Fi and Bluetooth Capabilities
* Applications of ESP32 in IoT Projects

### Practical Session

* Setting up ESP32 with PlatformIO
* Uploading programs to ESP32
* Using Serial Monitor
* Controlling LEDs
* PWM-based LED brightness control

## Activities Performed

1. Studied the architecture and features of ESP32.
2. Configured ESP32 development environment in PlatformIO.
3. Resolved package-related issues during setup.
4. Uploaded and tested multiple programs on ESP32.
5. Used Serial Monitor for debugging and communication.
6. Controlled LED using GPIO pins.
7. Implemented LED brightness control using PWM.

## Challenges Faced

* Faced package installation issues while configuring the environment.
* Initial setup felt chaotic due to multiple tools and configurations.
* Learned more about the PlatformIO project structure while troubleshooting.

## Learning Outcomes

* Understood the basics of ESP32 hardware and software architecture.
* Learned how Arduino-based programs are structured using setup() and loop().
* Gained hands-on experience uploading and testing programs on ESP32.
* Learned serial communication and PWM concepts.

## Summary

Today was my first practical session with ESP32. Although the setup process involved some troubleshooting, it helped me better understand the development workflow. I successfully uploaded multiple programs, used serial communication, and controlled LEDs using GPIO and PWM.
