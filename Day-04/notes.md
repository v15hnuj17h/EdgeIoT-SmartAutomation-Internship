# Development Environment Setup Notes

## Software Installed

### CP210x Universal Windows Driver

Used for communication between the computer and USB-to-UART devices such as ESP32 development boards.

### OpenJDK 25

Java Development Kit required by certain development tools and applications.

### .NET Desktop Runtime 8.0

Required for running applications built using the .NET platform.

### Visual Studio Code

Code editor used for development and project management.

### PlatformIO

An embedded development platform integrated with VS Code.

## Setup Steps

1. Install CP210x Driver.
2. Install OpenJDK.
3. Install .NET Desktop Runtime.
4. Install Visual Studio Code.
5. Install PlatformIO Extension from VS Code Extensions Marketplace.
6. Restart VS Code and verify PlatformIO installation.

## Key Learning

A properly configured development environment is essential before starting embedded systems and IoT projects.
