# Day 9 - KiCad PCB Design

## Date
26 June 2026

## Objective

Learn the complete PCB design workflow using KiCad, from schematic creation to generating manufacturing files.

## Topics Covered

- Introduction to KiCad
- Schematic Design
- Symbol Libraries
- Footprints
- PCB Layout
- Routing
- Fill Zones
- Silkscreen
- 3D Viewer
- Gerber File Generation

## Activities Performed

- Created a new KiCad project.
- Designed the circuit schematic.
- Assigned footprints to all components.
- Imported the schematic into PCB Editor.
- Placed and routed components.
- Added copper fill zones (Ground Plane).
- Used the silkscreen layer to print text on the PCB.
- Verified the PCB using design rule checks.
- Generated Gerber files.
- Compressed the Gerber files into a ZIP for PCB manufacturing.

## Learning Outcomes

- Learned the complete PCB design workflow.
- Understood the importance of footprints and routing.
- Learned how copper zones improve PCB performance.
- Generated manufacturing-ready Gerber files.

## Summary

Today's session focused on designing a PCB from scratch using KiCad. I completed the full workflow, including schematic design, PCB layout, routing, silkscreen editing, and generating Gerber files for fabrication.
