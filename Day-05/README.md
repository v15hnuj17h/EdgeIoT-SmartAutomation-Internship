# Day 5 - Applied Machine Learning Workshop

## Date

22 June 2026

## Objective

Build a simple machine learning model using a real-world loan approval dataset in Google Colab.

## Topics Covered

* Google Colab
* Dataset Loading
* Data Cleaning
* Data Preprocessing
* Decision Tree Classifier
* Model Evaluation
* Prediction using Custom Inputs

## Activities Performed

1. Loaded loan approval dataset from GitHub.
2. Cleaned missing values.
3. Converted categorical data into numerical values.
4. Split data into training and testing sets.
5. Trained a Decision Tree Classifier.
6. Evaluated model accuracy.
7. Visualized the decision tree.
8. Tested the model using custom user data.

## Learning Outcomes

* Understood the workflow of a machine learning project.
* Learned how to prepare data for training.
* Built and evaluated a classification model.
* Made predictions using a trained model.

## Summary

Today's session provided hands-on experience with machine learning using Google Colab. A Decision Tree model was trained on a loan approval dataset and used to make predictions based on user-defined inputs.
